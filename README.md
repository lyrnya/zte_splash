# zte_splash
***A tool for unpacking/repacking logo.bin***

`(UNISOC)`

# Usage
**unpack**
```
unpack.exe logo.bin
```
**repack**
```
repack.exe boot_logo.bmp boot_lowbatt.bmp chargelogo.bmp chargelogo2.bmp chargelogo3.bmp
```
***2025/11/14***
- `Runs well on the ZTE 7552N`
